## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

These are basic command and alog with it I have made the API live on vercel,
Base API url - https://stock-portfolio-be.vercel.app/portfolio/live/${USER_ID}
